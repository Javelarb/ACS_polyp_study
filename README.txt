# UCI ACS colorectal polyp study:

This repository include the OTU table, metadata, and R scripts used to analyze 16S, ITS, and Shotgun data produced from the pilot polyp study.  
Note: Please email javelarb@uci.edu to obtain access to the functional gene table and gene annotations, as they are too large for github.  

Also included are the shell scripts used to process the raw data.  

For 16S and ITS data, the processing parameters where recorded using the built in provenance tracking of qiime2's qza file format.
